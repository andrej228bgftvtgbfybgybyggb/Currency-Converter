# Currency-Converter
Currency Converter — это десктопное приложение на Python с графическим интерфейсом (Tkinter) для конвертации валют по актуальным курсам. Программа сочетает простоту использования с функциональностью: позволяет быстро пересчитывать суммы между валютами, видеть результат в реальном времени и отслеживать историю операций.
