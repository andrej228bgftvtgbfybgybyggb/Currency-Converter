# Currency Converter (Конвертер валют)

**Автор:** [Ваше Имя и Фамилия]

## Описание
GUI-приложение на Python (Tkinter) для конвертации валют в реальном времени, использующее API `exchangerate-api.com`. Сохраняет историю конвертаций.

## Как получить API-ключ
1. Зарегистрируйтесь на сайте [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Скопируйте свой ключ из личного кабинета.
3. Вставьте его в переменную `API_KEY` в файле `main.py`.

## Примеры использования
1. Введите сумму в поле "Amount".
2. Выберите исходную и целевую валюту из выпадающих списков.
3. Нажмите кнопку "Convert".
4. Результат появится под кнопкой, а запись добавится в таблицу истории.

## Тесты
- Проверка на 0: Приложение корректно обрабатывает сумму <= 0.
- Проверка на текст: Приложение вызывает ошибку при вводе букв (требуется доработка `try-except` для `ValueError`).
